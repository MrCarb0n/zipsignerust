# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within `zipsignerust`, please send an e-mail to Tiash / @MrCarb0n via `MrCarb0n@proton.me`. All security vulnerabilities will be promptly addressed.

Please **DO NOT** create a public GitHub issue for security vulnerabilities.
